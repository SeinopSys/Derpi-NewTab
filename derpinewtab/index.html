Settings
  • Filtering

    Rating tags

    Set the rating tags you want to allow when requesting images. Click to toggle.

    The image will be updated in to reflect the changes.

    Filters

    Filtering is controlled by whatever filter you have selected on derpibooru.org, or the Default filter if you aren't logged in. You have to enable/disable specific system tags above individually regardless of your filter setting.

  • Metadata

    Choose what information you want to see in the bottom right corner.

  • Image cropping

    This setting allows you to modify how the image is displayed if its size differs from the browser's size (which happens most of the time).
